Allow your BuddyPress community members to follow each other's activity.

== Description ==

BuddyPress Followers is a plugin that adds follow functionality to your BuddyPress community. Members can:

* Follow other members
* See who they're following
* See who follows them
* Get notifications when someone follows them

== Installation ==

1. Upload the `buddypress-followers` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure settings (if any) under Settings > BuddyPress > Follow

== Frequently Asked Questions ==

= How do members follow each other? =

Each member profile will have a "Follow" button that other members can click to follow them.

= Can I disable follow notifications? =

Yes, members can disable notifications in their notification settings.
